# yourenglishcoachtheme

WordPress theme build setup based on Vite and Sass.

## Requirements

- Node.js 16.x recommended (current project setup verified on 16.13.2)
- npm 8+

Notes:
- This project currently uses Vite 4.5.14 for compatibility with Node 16.
- Vite 8 requires Node 20.19+ or 22.12+.

## Install

```bash
npm install
```

## Development

```bash
npm run dev
```

## Production build

```bash
npm run build
```

Build output is generated in the `dist/` directory.
